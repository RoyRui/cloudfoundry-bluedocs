BlueDocs Documentation for CloudFoundry

The documentation contains information on two sets of data for CloudFoundry v1.0 and CloudFoundry v2.0 as collected by our BlueDocs message analysis tool:
  • NATS message content / publishers / subscribers to different NATS channels
  • Sequence of NATS message exchanges for each cf or vmc command as perceived by our analyzer tool

NATS Message Content

Message contents are captured by instrumenting the NATS client for each of the CloudFoundry components, intercepting in-going and out-going messages for each component, and analyzing the internals of the messages.
The documentation contains the following information:
  • The list of channels over which CloudFoundry components communicate
  • Content of messages exchanged over each channel, i.e., the list of keys in the message and their content types
  • List of publishers to each channel
  • List of subscribers to each channel

NATS Message Sequences

NATS message sequences contain informatioin on the set of applicable commands for cf or vmc and the series of messages exchanged during execution of each of those messages. Each sequence diagram captures the following information:
  • The source and the destination of a NATS message captured during execution of the CF command (shown with an arrow -->)
  • The channel over which the message is sent
  • The content of the message communicated over the channel
For better readability the message content and the source and target of each message exchange are hidden in the initial view of the diagram. In order to get the detailed view, the mouse can be hovered over the (+) sign or the (+) sign can be clicked to see the extended information.
